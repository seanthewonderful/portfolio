Freelance - oncoor® Website

oncoor® is a sports marketing agency specializing in representing college athletes.
I was contracted to make a splash page just to demonstrate some ideas I had for improving the website based on comments from oncoor's CEO. Soon thereafter I was coding the full web application for them will full functionality.
This site is build with Python and Flask, using Bootstrap styling with custom CSS and JQuery effects.
