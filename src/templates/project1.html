My first deployed, full-stack application

Using Python with Flask, this application allows users to view and review the restaurants around Disneyland theme park. This was my capstone project after 3.5 months of coding at Devmountain, and is also a passion project that I considered before I even knew how to code.
Full-stack web application coded with Python and Jinja2 on the Flask microframework.
Folium package with Leaflet.js used to create the maps. They are always up-to-date with the lates information in the database.
Bootstrap used for most of the styling elements, including making the application mobile-friendly.
PostgreSQL with SQLAlchemy ORM for database.
User login and logic created with Flask Login. Forms created with Flask WTForms.
Deployed with Heroku and connected to my Google Domain with SSL certiication.
