My first ever full-stack application

Using JavaScript and NodeJS, this is the first application I coded from start to finish. I was immensely proud of it, and even though it looks like the work of a strange person, I am still proud that I was able to accomplish this after just 7 weeks of learning.
Because I knew so little, I ironically coded almost every single element by myself from scratch. This explains inconsistencies and poor UX/UI design, but also demonstrates how much I was able to figure out without simply copy+pasting.
This project is not (yet) deployed because of quickly moving onto new projects, and at the time I built it locally without much knowledge of deployment possibilities.
GET/POST requests coded using Axios/Express.
Local JSON database allows users to add custom characters to the database, refreshed on server reset.
