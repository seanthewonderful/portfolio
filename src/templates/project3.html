My first interactive game

After learning about Tkinter, I learned how to create the interactive game, Snake!
This code is on my Github and I will attempt to package it with PyInstaller.
High Score is saved to a txt file so that it will always be remembered.
Though not thoroughly detailed, this was an exercise in understanding the principles of making a small, reactive game using code. It inspired me to explore more about video game programming as well as practical Tkinter usage.
